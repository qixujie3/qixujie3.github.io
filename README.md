# 看着你的时候，连手心里的汗水都写满了紧张


### 一、 [躲不掉的视线，掩饰不住的心跳](https://www.honeypeach.cloud)

有些喜欢，是捂住嘴巴也会从眼睛里跑出来的。每当我的视线里出现你的身影，周遭原本喧闹的世界仿佛瞬间被按下了静音键。我总是在人群中第一眼就捕捉到你，却又在你的目光投射过来的一刹那，慌乱地移开视线。

这种刻意的躲闪，不是因为不在意，而是因为心跳早已超速。我假装低头看手机，假装在和身边的朋友聊天，可眼角的余光却始终固执地锁定在你身上。那一刻，心口像是有无数只蝴蝶在振翅狂飞，撞得我胸口发闷，连呼吸都变得小心翼翼，生怕被你听见这急促而热烈的频率。

---

### 二、 [湿透的手心，写满年少的秘密](https://www.heartplay.cloud)

最出卖我的，其实不是闪躲的眼神，而是紧握的双拳。

每当你朝我走来，或者哪怕只是并肩站着，微风带过来你身上淡淡的洗衣粉清香，我的身体就会比大脑先一步做出反应。血液开始加速，一股温热的潮气迅速在掌心蔓延。

我常常下意识地把手放进口袋，或者在衣服上悄悄蹭掉那一层薄汗。可只要你一开口跟我说话，那抹潮湿就又会倔强地冒出来。那些密密麻麻、无法言说的紧张，全都被具象化成了手心里黏腻的汗水。如果汗水也有笔画，那它一定在我的掌心里，一遍又一遍地默写着你的名字，和一整个盛夏都藏不住的秘密。

---

### 三、[ 笨拙的对白，是最真挚的告白](https://www.moguc.cloud)

在没有面对你的时候，我在脑海里排练过无数次完美的开场白，想象自己能够幽默风趣、落落大方。可真正对上你眼睛的那一刻，那些精心准备的词汇瞬间碎成了一地浆糊。

“嗨……那个，今天天气挺好的。”
听到自己说出这么蠢的话，我简直想找个地缝钻进去。因为紧张，我的声音带着一丝微不可察的颤抖，手指更是死死地攥着衣角，手心里的汗水把布料都浸染得有些发潮。

可你却看着我笑弯了眼，温柔地回应着我的笨拙。那一刻我突然明白，这种连手心出汗都无法控制的紧张，虽然让我看起来不够完美、甚至有些滑稽，但它却是我在这个年纪里，能够给你的、最毫无保留也最真挚的喜欢。
